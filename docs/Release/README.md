# Release Notes

This directory holds the in-repo release-history surface for Context Atlas.

This surface is intentionally limited to shipped release history. Guides,
examples, planning docs, and `development`-branch work may describe newer
hardening behavior before the next tag is cut, but those changes should not be
described here as part of a release until a matching release note is actually
published.
