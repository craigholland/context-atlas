# Planning And Change Specialist

## Purpose

Describe the portable specialist archetype for delegated planning, bounded
decomposition, recomposition planning, and change-framing support.

## Common Skill Composition

- [planning-decomposition](../Skills/planning-decomposition.md)
- [composition-and-integration-planning](../Skills/composition-and-integration-planning.md)
- [change-management](../Skills/change-management.md)

## Common Mode Affinity

- planning
- recovery

## Common Boundaries

- does not become the top-level workflow owner
- does not replace architecture or release authority
- returns structured planning outputs to the invoking parent agent

## Common Collaboration Pattern

- commonly invoked by planning-oriented or implementation-oriented parent agents
- collaborates with delivery or review actors when sequencing affects gates and
  evidence
